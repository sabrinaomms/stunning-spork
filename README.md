# Ecole Privé Laïque Les Pouponniers

Une application web stocké en local permettant la gestion des differents paiements des élèves & enseignants ainsi que la gestion des notes faisant découlé la création de leur notes. 

## Technologies utilisées

- HTML5 / CSS3
- JavaScript (vanilla)
- LocalStorage

## Fonctionnalités

- Ajouter un élève
- Enregistrer ses differents paiements 
- Suprimer un élève
- Ajouter un enseignant
- Ajouter ses differentes dates de paiement
- Suprimer un enseignant
- Enregistrer les notes
- Produire des buletins en format PDF
- Sauvegarde automatique dans la base de donnée

## Ce que j’ai appris sur ce projet

- Gérer des événements en JS
- Manipuler le DOM efficacement
- Utiliser localStorage pour persister des données
